# Elections Scraper

Projekt slouží ke scrapování výsledků voleb z webu volby.cz pro vybraný územní celek a k uložení výsledků do CSV souboru.

## Instalace

Nejprve vytvoř a aktivuj virtuální prostředí:

python -m venv .venv

.venv\Scripts\activate

instalace modulů:
pip install -r requirements.txt
pip install requests beautifulsoup4
pip install requests
pip install bs4
pip freeze > requirements.txt
